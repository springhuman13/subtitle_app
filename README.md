# 📝 Subtitle App

A Python application for automating subtitle processing.

The project simplifies repetitive subtitle editing tasks and helps prepare subtitle files for further use.

---

## Features

- Subtitle processing
- File manipulation
- Automation of repetitive tasks

---

## Tech Stack

- Python

---

## Getting Started

```bash
pip install -r requirements.txt

python main.py
```
